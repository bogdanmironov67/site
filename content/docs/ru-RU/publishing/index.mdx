title: Публикация мода
description: Узнайте, как опубликовать свой мод для сообщества.
---

Теперь, когда мод готов, его можно опубликовать и показать сообществу. Существуют разные площадки для публикации модов, например Modifold, Modtale, CurseForge и BuiltByBit.

Ниже собраны руководства по публикации на разных площадках, чтобы можно было быстро выбрать нужный сервис и подготовить страницу проекта.

- [Публикация на Modifold](./publishing/modifold)
- [Публикация на Modtale](./publishing/modtale)
- [Публикация на CurseForge](./publishing/curseforge)
- [Публикация на BuiltByBit](./publishing/builtbybit)

<Callout>
  Если вы представляете площадку для публикации модов, напишите на `neil@hytalemodding.dev`, чтобы добавить её в документацию.
</Callout>
