# Публикация мода на Modifold

[Modifold](https://modifold.com) - это платформа для моддинга, где авторы могут загружать свои моды и делиться ими с сообществом.

Это руководство повторяет реальный процесс публикации проекта: от входа в аккаунт до отправки на модерацию.

![](/assets/guides/modifold_home.png)

## Шаг 1. Вход

Сначала нужно войти в аккаунт.  
В шапке сайта нажмите **"Войти"** и выберите удобный способ входа.

Сейчас доступны:
- GitHub
- Discord
- Telegram

![](/assets/guides/modifold_auth.png)

## Шаг 2. Создание проекта

После успешного входа нажмите **"Создать проект"** в шапке сайта.

Заполните:
- **Название проекта**
- **Краткое описание** (краткое описание в 1-2 предложениях, например: `Этот проект добавляет...`)

Затем нажмите **"Создать проект"**.

После создания сразу откроется страница настроек проекта.

![](/assets/guides/modifold_project_settings_general.png)

## Шаг 3. Иконка проекта

Сначала рекомендуется загрузить иконку проекта.

Лучше всего использовать внутриигровой скриншот мода, чтобы пользователи сразу понимали, что делает проект.  

Загрузка иконки не обязательна, но очень рекомендуется.

## Шаг 4. Описание

Обязательно заполните **Описание** проекта.

[Modifold](https://modifold.com) поддерживает удобный редактор Markdown, в котором можно подробно описать:
- что делает мод
- как им пользоваться
- важные детали для пользователей

Баннеры со ссылками на другие платформы разрешены.  
Минимальная длина описания: **50 символов**.

![](/assets/guides/modifold_project_settings_description.png)

## Шаг 5. Загрузка версии мода

Это самый важный этап.

Загрузите файл версии проекта (например, `.jar`) через блок загрузки:
- перетащите файл мышкой, или
- нажмите на блок и выберите файл на компьютере

![](/assets/guides/modifold_project_settings_upload_version.png)

Затем заполните:
- **Номер версии** (например, `1.0.1`, если это исправление ошибок в моде)
- **Список изменений** (необязательно, в формате Markdown)
- **Канал релиза**: `Релиз`, `Бета` или `Альфа`

После этого нажмите **"Загрузить версию"**.

Готово, первая версия загружена!

<Callout type="info">
Если мод публикуется на Modifold, страницу проекта стоит поддерживать в актуальном состоянии, чтобы пользователи быстро понимали, что делает мод и какую версию им нужно устанавливать.
</Callout>

## Шаг 6. Галерея

Во вкладке **"Галерея"** (рекомендуется) можно загрузить изображения, показывающие возможности мода.

Одно изображение можно отметить как **Рекомендуемое**.

Выбранное изображение галереи показывается в результатах поиска и на карточке проекта.

Оно будет использоваться как обложка проекта.

## Шаг 7. Ссылки

Во вкладке **"Ссылки"** можно добавить полезные ссылки проекта:
- Трекер ошибок
- Исходный код
- Вики (ссылка на внешние ресурсы)
- Вики HytaleModding
- Discord

Эти ссылки отображаются отдельным блоком на странице проекта.

![](/assets/guides/modifold_project_settings_links.png)

## Шаг 8. Теги

Во вкладке **"Теги"** (рекомендуется) выберите подходящие теги проекта.  
Например, если мод добавляет декоративные элементы, можно выбрать тег `Decoration`, а затем нажать **"Сохранить"**.

## Шаг 9. Лицензия

Во вкладке **"Лицензия"** можно указать лицензию проекта.

По умолчанию используется **ARR (All Rights Reserved)**, но можно выбрать и другую доступную лицензию.

## Шаг 10. Модерация

Последний обязательный этап - вкладка **"Модерация"**.

Там можно проверить, что все обязательные поля заполнены и отправить проект на проверку.

![](/assets/guides/modifold_project_settings_moderation.png)

После отправки модераторы проверяют:
- соответствие правилам контента Modifold
- версии проекта на наличие вредоносного кода

Когда проверка завершится, придёт уведомление:
- если проект одобрен, он станет публичным
- если проект отклонён, модераторы укажут причину, после чего проект можно отправить повторно после исправлений

# Интеграция с HytaleModding Wiki

Если мод использует [HytaleModding Wiki](https://wiki.hytalemodding.dev), достаточно добавить ссылку на вики (в поле **Вики HytaleModding**) в настройках проекта Modifold.

После этого у проекта появится вкладка **"Вики"** с синхронизированными страницами и Markdown-контентом.

Настройка и редактирование вики остаются на сайте [HytaleModding Wiki](https://wiki.hytalemodding.dev).

![](/assets/guides/modifold_project_wiki.png)

На скриншоте выше показана вики-страница проекта Voile: [modifold.com/mod/voile/wiki](https://modifold.com/mod/voile/wiki)

# Получение помощи

Если при загрузке проекта возникли проблемы:

- обратитесь в официальный Discord-сервер Modifold
- или отправьте письмо на `support@modifold.com` с описанием проблемы
